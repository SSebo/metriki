# Metriki

Metriki is a rust library ported from [Dropwizard Metrics](https://github.com/dropwizard/metrics).

This library is a work in progress.

## Features and TODOs

- Metrics
  - [x] meter
  - [x] histogram
  - [x] timer
  - [x] gauge
  - [x] counter
- Reporters
  - [x] logger
  - [ ] influxdb
  - [ ] riemann
- Instruments
  - [x] warp
  - [ ] reqwest
